# 公子 · 公司工资管理系统

本地单页工资管理应用，数据保存在浏览器本地（localStorage），无需后端即可使用。

- **在线预览**：[gongzi-zeta.vercel.app](https://gongzi-zeta.vercel.app)
- **更新日志**：[CHANGELOG.md](./CHANGELOG.md)

## 功能概览

| 模块     | 说明 |
|----------|------|
| 工资表   | 按月录入/编辑，出勤与扣款自动计算，导出 CSV、打印 |
| 员工管理 | 员工档案与基础工资，可同步到工资表 |
| 基数设置 | 社保/公积金基数与比例、发薪日 |
| 历史工资 | 按月份只读查看、导出该月 CSV |
| 考勤     | 从企业微信打卡月报 CSV 导入出勤天数并同步到工资表 |
| 查询     | 按姓名查各月工资、导出该人汇总 CSV |
| 工资条   | 按月份生成每人一条工资条，支持按人筛选与打印 |
| 成本汇总 | 按月汇总应发/社保/个税/实发，按年筛选，点击月份跳转工资表 |
| 数据备份 | 备份到文件 / 从文件恢复、云端备份（JSONBin.io） |

## 使用说明

1. 打开 `index.html` 或部署后的页面即可使用，无需安装。
2. 数据仅保存在当前浏览器，请定期使用「数据备份」导出或云端备份。
3. 考勤可从企业微信管理后台导出月报（CSV）后，在「考勤」页选择月份并导入。

## 技术说明

- 单文件 HTML + CSS + JavaScript，无构建步骤。
- 本地存储键：`localPayrollData`、`localPayrollEmployees`、`payrollSocialBases`、`payrollSocialRates` 等，详见代码。
